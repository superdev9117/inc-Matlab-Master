What type of looping structure?

  • For
  • While
x = [2 -3 -5 0 -3];

count = 0;
for index = 1: length(x)
   if x(index) >= 0
   count = count + 1;
   end
end

What type of conditional structure?

  • If
  • Switch
a = randi(100, 1);

if a < 30
   disp('small')
elseif a < 80
    disp('medium')
else
   disp('large')
end
